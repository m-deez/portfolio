Alley Scoop


Myself and a team of two others created Alley Scoop to be the go-to source of sports news. And to be able to share and comment on sports articles. We did a large amount of crowd poling to find out what a sports enthusiast would want to see in a social app like this. None of us knew much about sports. But we wanted to challenge ourselves to code outside our comfort zones.

We created a "like/dislike" function in the form of a score board, made a way to add comments to any post, and implemented a news feed API. Unfortunately the news API we used is only free in developer mode and you won't be able to see/use it when you visit the sight.

TECH: JavaScript | React | HTML | CSS | Heroku | API | Node.js | EJS | Mongoose
